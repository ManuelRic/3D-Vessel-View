General:
    right click hold + rotate => rotate camera

On Vessel:
    hover     => show name
    click     => show boat detail card
    dbl click => same as click + vessel focus

On Empty:
    click     => restore to default (close details + focused elements) ?
    drag      => move camera around map

On Trail:
    hover     => show corresponding vessel name
    click     => elevate trail y axis + highlight trail
    dblclick  => same as click + dim other trails
